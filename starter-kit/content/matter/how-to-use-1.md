---
title: "How to use this kit"
lede: "Three rules: carry it, show the right page, keep everything signed."
---

**1. Carry it.** Fold it in half along the spine and keep it in the same bag as your insurance card and photo ID.

**2. Show the right page.** Each guide page has a code (for example `UPCJ/SK/1.3/EN`). That code matches the step you are on in the journey map on page 4. When a clerk asks what you need, show them the page.

**3. Keep everything signed.** Every template page has a stamp box. If a page you hand to the hospital comes back without a signature or stamp, it is not a valid document. Ask for it to be re-signed before you leave the counter.
